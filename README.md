# 🤖 BotLeague

A responsive frontend implementation developed as part of the **Full Stack Developer Intern assignment** for **BotMakers Pvt Ltd**.

[Figma Design Reference](https://www.figma.com/design/9iGNHKi7vwCCnPrBLwwQxc/AI-Shopping-Assistant?node-id=52-2&t=nmiYgJNxr0RVwGiY-1) | [Live Demo](https://botleague-seven.vercel.app/)

---

## ✨ Features

- Responsive design across desktop, tablet, and mobile devices
- Mobile navigation menu with smooth transitions
- Reusable and data-driven React components
- TypeScript interfaces for structured component data
- Interactive hover and touch effects
- Custom gradients and styling based on the provided Figma design
- Optimized layouts for different screen sizes

---

## 🛠️ Tech Stack

- **React**
- **TypeScript**
- **Tailwind CSS**
- **Vite**

---

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/mannatgupta146/botleague.git
```

Navigate to the project directory:

```bash
cd botleague
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.
